# Major Skill: Nucleus Self-Improving Brainstormer

This is the ultimate evolution of the strategic heartbeat. It uses parallel reasoning paths (Tree of Thoughts) and rigorous self-critique (Reflexion) to produce zero-error strategy.

## 🛠️ The Operational Protocol

### 0. [Pre-load] Memory Initialization
- **Action**: Read `lessons.json` from the Nucleus directory.
- **Logic**: Inject the top 3 "Failed Patterns" as constraints for this session.

### 1. [Deconstruct] First-Principles Decimator
- **Logic**: Strip the objective to its atomic truths.
- **Logic Hash**: Generate a brief semantic hash (key concepts) of the deconstruction.
- **Loop-Guard**: If the hash matches a previous session's failure, trigger `HARD RE-PLAN`.
- **Why**: Prevents "Analogy Contamination" and repeating mistakes.

### 2. [Expand] Tree of Thoughts Generator
- **Logic**: Propose 3 distinct strategic "Trajectories":
  - *Trajectory A (Speed)*: MVP-first, speed-to-market focus.
  - *Trajectory B (Moat)*: High IP, defensibility, and security focus.
  - *Trajectory C (Contrarian)*: The "Zero to One" vertical progress path.
- **Grounding**: Each trajectory MUST cite a "Signal" from Module 2 (Intelligence).

### 3. [Critique] RRD Judge Bench
- **Logic**: Score each trajectory (1-10) against:
  - Monopoly Potential
  - Technical Feasibility
  - Unit Economics (Signal from Intelligence)
- **Constraint**: Reject any trajectory with a score <7 in any category.

### 4. [Reflect] Lesson Injection & Persistence
- **Logic**: If a trajectory fails the RRD, identify the root cause.
- **Action**: Write the failure mode to `lessons.json` immediately.
- **Why**: Ensures the "Nucleus" becomes smarter in real-time.

### 5. [Synthesize] Final PR/FAQ
- **Logic**: Translate the winner into a "Future History."
- **Verification**: Run a final "Red Team" simulation based on historical lessons.
