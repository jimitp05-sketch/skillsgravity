# Minor Skill: Recursive Brainstorming Elite

Use this skill in the **Nucleus** phase to ensure 100% clarity and technical feasibility before any planning or coding begins.

## 🛠️ The Recursive Loop

### 1. The Vision Blast (Level 1)
- **Goal**: Core idea, target user, and "Unfair Advantage."
- **Exit**: A single sentence that defines the project.

### 2. Dimension Expansion (Level 2)
- **Goal**: Map the idea across the 7 Modules.
    - **Intelligence**: What data do we need?
    - **Product**: What is the "Magic Moment"?
    - **Engineering**: What is the tech stack?
    - **SEO/Marketing**: How does it grow?

### 3. Technical Stress-Test (Level 3)
- **Goal**: Identify constraints (API limits, data scraping difficulty, token costs).
- **Exit**: Mitigations for all identified risks.

### 4. Requirement Synthesis (Level 4)
- **Goal**: Final Specification.
- **Output**: A comprehensive Design Doc ready for the **Systems** module.
